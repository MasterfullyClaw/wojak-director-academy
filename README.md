# Wojak Director Academy

Interactive quest game that teaches you how to create cinematic Wojak-style AI videos — from story concept to final video.

**[Play Now →](https://masterfullyclaw.github.io/wojak-director-academy/)**

## 6 Levels

| Level | Name | What You Learn |
|-------|------|---------------|
| 1 | The Concept | Story planning — how to define your video in one sentence |
| 2 | The Breakdown | Scene structure — one action per scene, camera angles, expressions |
| 3 | The Lock | Consistency — wardrobe lock, recurring motifs, time patterns |
| 4 | The Prompt | Prompt anatomy — what's locked vs what you fill in |
| 5 | The Clip | Video prompts — shots, camera motion, color palettes |
| 6 | The Choice | Manual vs Auto — choose your workflow route |

## Two Routes

After completing the game, choose how you create:

- **Manual Route** — Use prompt templates. Fill in 5 blanks per scene. Paste into any AI image tool.
- **Auto Route** — [Wojak Director GPT](https://chatgpt.com/g/g-69d59f1df66c8191b3eb4af278b6df24-wojak-director) guides you through the entire workflow conversationally.

## Tech

Pure vanilla HTML/CSS/JS. No frameworks. No build tools. Hosted on GitHub Pages.
